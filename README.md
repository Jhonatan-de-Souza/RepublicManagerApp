# Republic Manager App

This app will help you:

1. Share Groceries Expenses
2. Share Daily Taks
3. Manage Expenses in a shared Location
4. Create Profiles for "Republics(any share location)" for administrator and regular users

# Made with

1. Angular 7+
2. Nativescript
3. Javascript ES6+
4. Html
5. CSS
